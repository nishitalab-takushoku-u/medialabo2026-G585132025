「WEBコンテンツ作成」の成果

本テーマについて

このテーマでは、HTML,CSS,JavaScriptの基本を学び、これらを組み合わせてWebページを構成する方法を学びます。目標としてはWebページの構造を記述する基本的な方法や,Webページの見て目の設定の基本的な方法などがわかるなどが挙げられます。

プロジェクト課題について

この科目を通して学んだHTML,CSS,JavaScriptをもとにプロジェクト課題に取り組みました。個人的に、HTMLを作ってCSSを作ったところまでは順調に進んでたと感じていましたが、HTMLで書いたものをJavaScriptに移し替える作業にすごく苦戦しました。JavaScriptに入ってから基本的なコードがとても多くなって、似てるものも多く使い分けがうまくできていなかったので苦戦しました。

各授業の学習事項

第1回 WWW と HTML

第1回目の授業で学んだことは、WWWとHTMLですがHTMLの少し詳しく学びました。具体的には見出しの作り方やリストの作り方、画像の添付の仕方や、ハイパーリンクの作り方などです。自分的にはハイパーリンクに少し苦戦しました。

演習の成果: 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

第2回目で学んだことはCSSでWEBページをどのように装飾するかということです。具体的には、文字の色の変え方や、フォントの変え方などです。

演習の成果: 演習2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

第3回目の授業で学んだことは、JavaScriptのすごい基本的なところを学びました。JavaScriptでできることや、JavaとJavaScriptの違いや、簡単な構文の使い方、デバックの仕方などです。癖でJavaの構文を使わないように気をつけたいです。

演習の成果: 演習3-6: オブジェクトの参照

第4回 DOM と Javascript

第4回目ではフォームの要素、イベントの処理、DOMO(Document Object Model)について学びました。この回からフォームで入力などサイトを見る人ができることが多くなったと個人的に思いました。

演習の成果: 演習4-10: 足し算ページ

第5回 イベント処理

第5回の授業では、新たに5つの構文を学びました。1つ目は要素の新規作成、2つ目は要素1の内部に要素2を配置するというもの。3つ目は内部ではなく直後に配置するもの。4つ目は、要素の属性を設定、5つ目は要素を削除するものです。

演習5-7: DOMプログラム(2)
